# Chart 01: Schema Null Rates

## What this shows
The missingness rate across key procurement record attributes.

## Why it matters
Determines features suitable for downstream machine learning and validates schema completeness.

## Key findings
- Most columns have very low null rates (under 5%), indicating high completeness.
- Buyer/supplier names and identifiers are fully populated (0% null rate).
- Contract values and final prices are clean, showing data entry reliability.

## Limitations
Does not explain why some fields are blank (e.g., optional fields in specific procedures).
