# Section C: Supplier Ecosystem Summary

## Purpose
Profiles supplier sizes, power-law distributions, top suppliers, and year-on-year vendor activity.

## SQL-Backed Charts
- `09_top_20_suppliers_by_contract_count`: Chart 09: Top 20 Suppliers by Contract Count
- `10_supplier_contract_count_distribution`: Chart 10: Supplier Contract Count Distribution
- `11_supplier_yoy_activity`: Chart 11: Supplier Year-on-Year Activity

## Artifacts
PNG, HTML, Markdown, and metadata JSON files are generated and stored in `results/eda/plots/` for each chart.
