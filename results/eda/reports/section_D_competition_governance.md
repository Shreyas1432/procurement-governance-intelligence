# Section D: Competition & Governance Summary

## Purpose
Analyzes procedure selections, single-bid rates, bid count distributions, and annual competition trends.

## SQL-Backed Charts
- `12_procedure_type_distribution`: Chart 12: Procedure Type Distribution
- `13_single_bidder_rate_by_procedure_type`: Chart 13: Single-Bidder Rate by Procedure Type
- `14_bid_count_distribution`: Chart 14: Bid Count Distribution
- `15_single_bidder_rate_trend_by_year`: Chart 15: Single-Bidder Rate Trend by Year

## Artifacts
PNG, HTML, Markdown, and metadata JSON files are generated and stored in `results/eda/plots/` for each chart.
