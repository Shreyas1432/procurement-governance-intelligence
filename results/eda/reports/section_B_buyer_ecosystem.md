# Section B: Buyer Ecosystem Summary

## Purpose
Evaluates buyer classification, total spend concentration, HHI patterns, and top supplier dependence.

## SQL-Backed Charts
- `05_buyer_count_by_type`: Chart 05: Buyer Count by Type
- `06_buyer_spend_distribution`: Chart 06: Buyer Spend Distribution
- `07_buyer_dependency_ratio_distribution`: Chart 07: Buyer Dependency Ratio Distribution
- `08_hhi_distribution_across_buyers`: Chart 08: HHI Distribution Across Buyers

## Artifacts
PNG, HTML, Markdown, and metadata JSON files are generated and stored in `results/eda/plots/` for each chart.
