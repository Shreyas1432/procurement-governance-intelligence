# Success Criteria

## RQ1

- Modularity > 0.3
- Resilience standard deviation > 0.15
- Dependency-single-bidder correlation is computed and reported

## RQ2

- Random Forest weighted F1 >= 0.70
- XGBoost/fallback boosting AUC >= 0.75
- Logistic Regression AUC >= 0.65

## RQ3

- Gradient Boosting R2 >= 0.30
- RMSE reduction >= 15 percent vs CPV baseline
- Consensus anomaly rate is monitored
