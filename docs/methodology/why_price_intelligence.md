# Why Price Intelligence

Price intelligence is meaningful only when contracts are compared within plausible categories. CPV-constrained modeling reduces the risk of flagging a normal high-cost category as anomalous simply because it is expensive.

Gradient boosting regression estimates expected log price from category, procedure, bid count, estimate, and time. Isolation Forest and Local Outlier Factor add model-agnostic anomaly evidence. Consensus flags are used because a single anomaly method can overreact to distribution shape.

The output is an abnormal-pricing signal for governance review, not a fraud claim.
