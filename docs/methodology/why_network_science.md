# Why Network Science

Procurement governance depends on relationships, not only individual contracts. Network science models buyers and suppliers as a bipartite system so the project can measure concentration, dependence, community structure, and resilience.

For RQ1, centrality identifies structurally important suppliers and buyers. For RQ2, network position becomes a governance context signal. For RQ3, central suppliers or dependent buyers can be compared against price anomaly patterns to test whether abnormal prices cluster around structurally powerful actors.

The main control is interpretability: centrality is used as evidence of structural position, not as proof of misconduct.
